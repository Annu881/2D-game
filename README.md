# Neon Shift

A polished 2D platformer with gravity-switching mechanics made in Godot 4.

## How to Run

1.  **Download Godot 4.x** (Latest stable version recommended).
2.  **Import Project**:
    *   Open Godot.
    *   Click "Import".
    *   Navigate to this folder and select `project.godot`.
3.  **Play**:
    *   Press F5 to run the project.
    *   Or open `scenes/MainMenu.tscn` and press F6.

## Controls

*   **Move**: Arrow Keys or WASD
*   **Jump**: Space / Enter / Z
*   **Flip Gravity**: Tab / Shift / X

## Features

*   **Gravity Flip**: Switch gravity up and down to navigate levels.
*   **Hazards**: Avoid spikes and navigate tricky platforms.
*   **Collectibles**: Gather coins to increase your score.
*   **Level Progression**: Reach the goal to advance (currently loops or restarts).

## Project Structure

*   `scenes/`: Contains all game scenes (Player, Level, UI).
*   `scripts/`: Contains all GDScript logic.
*   `assets/`: (Placeholder) Graphics and audio.
